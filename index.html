Projeto em Destaque

Hackathon DBC — Semana de Acessibilidade

Durante o hackathon desenvolvemos uma página para promover a segunda semana de acessibilidade da DBC. Eu atuei principalmente no backend (Java + Spring Boot), integrando APIs, criando endpoints para cadastro de usuários e eventos, e garantindo segurança básica e performance. O frontend em React oferecia diferentes visões (admin, usuário), recursos de acessibilidade (ajustes de contraste, leitura por voz, navegação por teclado) e formulários para inscrição em eventos. O projeto foi uma entrega colaborativa com foco em inclusão digital.
